# TaskNotes

**GitHub:** https://github.com/liamcain/obsidian-task-notes

## What is the TaskNotes Plugin?

**TaskNotes** transforms Obsidian into a **powerful task management system** that lives in your notes. It lets you:

- ✅ Create, track, and manage tasks within your vault
- 📅 Set due dates, reminders, and recurring tasks
- 🏷️ Tag and prioritize tasks for better organization
- 📊 View tasks across multiple notes at once
- ⚡ Quick-capture tasks without leaving your current note
- 🔍 Filter and search tasks by status, date, or tag
- 🎯 Track task completion and see your progress

## Why Use TaskNotes?

**TaskNotes keeps your tasks where your knowledge lives—in Obsidian:**

- **Integrated workflow** — No switching between apps; tasks live in your notes
- **Context-aware** — Tasks stay connected to the notes they relate to
- **Flexible** — Use your preferred task syntax and customize to your workflow
- **Discoverable** — Search and filter tasks alongside your other notes
- **Progress tracking** — See what you've accomplished at a glance
- **Anxiety relief** — Never lose a task; everything is backed up and searchable
- **Single source of truth** — Your knowledge and your tasks in one place

## Key Features at a Glance

| Feature | What It Does |
|---------|------------|
| **Task Creation** | Write tasks using markdown syntax (`- [ ] Task name`) |
| **Due Dates** | Set deadlines with `due: YYYY-MM-DD` in task metadata |
| **Priorities** | Mark tasks as high, medium, or low priority |
| **Task View** | See all tasks across your vault in one dashboard |
| **Filtering** | Filter by status, date, priority, tag, or custom criteria |
| **Recurring Tasks** | Create tasks that repeat daily, weekly, monthly |
| **Task Notes** | Add detailed notes and descriptions to individual tasks |
| **Completion Tracking** | Mark tasks done and see completion statistics |

## How It Works

1. **Write tasks naturally** — Use standard markdown task syntax in your notes
2. **Add metadata** — Optionally include due dates, priorities, tags
3. **View everywhere** — Tasks appear in your notes AND in the task dashboard
4. **Filter & search** — Find tasks by any criteria you need
5. **Complete tasks** — Mark done and watch your progress grow

## Task Syntax Examples

**Basic task:**
```markdown
- [ ] Review budget proposal
- [x] Send monthly report (completed)
```

**Task with metadata:**
```markdown
- [ ] Finish project writeup @home #work due: 2026-07-15 priority: high
- [ ] Review PR comments @computer due: 2026-07-12 
- [x] Schedule team meeting #meeting
```

**Nested tasks:**
```markdown
- [ ] Q3 Planning
  - [ ] Review Q2 results
  - [ ] Set quarterly goals
  - [x] Budget planning
```

## Example Workflow

**In your daily note:**
```markdown
# 2026-07-10

## Today's Priorities
- [ ] Finish presentation #work due: 2026-07-10 priority: high
- [ ] Email client with update #urgent
- [ ] Team standup at 10am #meeting

## Focus Time
- [ ] Deep work on research project #research
```

**What happens:**
- ✅ Tasks appear in your daily note
- ✅ They also show up in your Task Dashboard
- ✅ You can filter to see only "high priority" or "due today"
- ✅ Cross-reference tasks with related notes using links
- ✅ Mark complete and celebrate your wins!
