# Daily Notes

**GitHub:** https://github.com/liamcain/obsidian-daily-notes-plugin

## What is the Daily Notes Plugin?

**Daily Notes** is a foundational Obsidian plugin that enables **automatic daily note creation** with customizable templates and folder structures. It lets you:

- 📅 Create a new daily note with one click or keyboard shortcut
- 📝 Automatically use templates for consistent daily structure
- 🗂️ Organize daily notes by folder (daily, 2024, 2026, etc.)
- 🔗 Easily navigate between days (previous/next day)
- ⏰ Insert the current date in any format you prefer
- 📖 Build a daily journal or log of your life
- 🔍 Quick access to today's note from anywhere in the vault
- 📊 Foundation for building daily practices (journaling, planning, reflection)

## Why Use Daily Notes?

**Daily Notes creates the backbone of a personal knowledge system:**

- **Capture everything** — One place for today's thoughts, tasks, and learnings
- **Build a journal** — Over time, a record of your life unfolds
- **Consistency** — Same structure daily reduces friction
- **Quick access** — Open today's note instantly with a hotkey
- **Reflection practice** — Daily notes encourage reflection and growth
- **Memory aide** — Capture what happened, what you learned, how you felt
- **Goal tracking** — Monitor daily progress on important projects
- **Writing habit** — Daily notes practice turns into stronger writing

## How Daily Notes Works

1. **Configure the plugin** — Set folder location, naming format, template
2. **Create shortcut** — Assign hotkey (e.g., Cmd+Shift+D)
3. **Press hotkey** — Daily note opens instantly
4. **Use template** — Pre-filled structure guides your entry
5. **Write freely** — Capture your day, thoughts, tasks, learnings

## Example Workflow: Daily Routine

**Morning (5 minutes):**
```
1. Press Cmd+Shift+D to open today's note
2. Identify 3 priorities for the day
3. Check schedule
4. Skim yesterday's note for carryovers
```

**Throughout the Day (2-3 minutes every few hours):**
```
1. Jot down important insights
2. Log meeting notes
3. Note wins and blockers
4. Capture learnings
```

**Evening (10 minutes):**
```
1. Complete reflection section
2. Review day's accomplishments
3. Note tomorrow's prep
4. Capture wins for motivation
```

## Tips for Daily Note Success

💡 **Create a template** — Use Templater plugin to auto-populate structure
💡 **Set a hotkey** — Make opening today's note effortless
💡 **Write freely** — No pressure for perfect prose; capture authentically
💡 **Include wins** — Celebrate daily accomplishments, however small
💡 **Link to notes** — Create connections to relevant projects and ideas
💡 **Be consistent** — Even 5 minutes daily compounds over time
💡 **Review weekly** — Sunday review of the week's entries reveals patterns
💡 **Archive old notes** — Keep recent notes accessible, older ones organized
