# Link Converter

**GitHub:** https://github.com/ozntel/obsidian-link-converter

## What is Link Converter?

**Link Converter** is an essential Obsidian plugin that transforms links between different formats with a single command. It lets you:

- 🔄 Convert wiki-links to markdown links (and vice versa)
- 📝 Transform markdown links to reference-style links
- 🎯 Switch between absolute and relative paths
- 🚀 Bulk-convert links across your entire vault
- ✨ Maintain full control over which format you prefer

## Why Use Link Converter?

**Link Converter solves a common problem:**

- You might start with markdown links `[text](path)` but later prefer wiki-links `[[note]]`
- Converting manually is tedious and error-prone
- Different note formats may require different link styles
- Migrating old vaults to Obsidian often means converting external markdown links
- Standardizing your vault's link format keeps things clean and consistent

## Key Features at a Glance

| Feature | What It Does |
|---------|------------|
| **Wiki-link ↔ Markdown** | Switch between `[[note]]` and `[note](path)` formats |
| **Heading Links** | Convert `[[note#heading]]` between formats |
| **Image Links** | Transform image link formats consistently |
| **Path Types** | Convert between absolute and relative paths |
| **Bulk Convert** | Apply conversions to the entire vault at once |
| **Smart Detection** | Automatically detect which format to convert |
| **Undo Support** | Safely revert conversions if needed |

## How It Works

1. **Open a note** with links you want to convert
2. **Use the command** — Run "Link Converter: Convert links in current file"
3. **Choose the format** — Pick your target format (wiki-link, markdown, etc.)
4. **Done!** — All links in that note are instantly converted

Or convert your **entire vault** with "Link Converter: Convert all links in vault"

## Example: Converting Link Formats

### Before (Markdown Links)
```markdown
See [my project](project-notes/my-project.md) for details.

Read the [team guide](docs/team-guide.md#procedures) here.

Check out ![diagram](assets/architecture.png)
```

### After (Wiki-Links)
```markdown
See [[project-notes/my-project]] for details.

Read the [[docs/team-guide#procedures]] here.

Check out ![[architecture.png]]
```

**All converted with one click!**
