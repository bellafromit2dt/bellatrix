# Make

**Website:** https://www.make.com/
**Plugin Documentation:** Obsidian Make integration

## What is the Make Plugin?

**Make Plugin** connects Obsidian to **Make.com** (formerly Integromat), unlocking powerful automation and integration capabilities. It lets you:

- 🔗 Trigger Make.com scenarios from within Obsidian
- 📥 Auto-import data from external services into your notes
- 🚀 Automate repetitive tasks across tools
- 🔄 Create two-way syncing with other apps
- 📊 Send data from Obsidian to other platforms
- 🤖 Build complex automation workflows
- ⚡ No-code automation accessible to everyone
- 🌐 Connect hundreds of apps without leaving Obsidian

## How the Make Plugin Works

1. **Design a scenario in Make.com** — Set up your automation workflow
2. **Get webhook/API link** — Make provides integration details
3. **Configure in Obsidian** — Link Make Plugin to your scenario
4. **Trigger from notes** — Use commands or buttons to start automations
5. **Watch data flow** — Obsidian ↔ Make.com ↔ Other apps

