# Git Integration for Obsidian

**GitHub:** https://github.com/denolehov/obsidian-git

## What is the Obsidian Git Plugin?

**Obsidian Git** is a powerful plugin that brings **version control** directly into your vault. It lets you:

- 💾 Commit changes to your notes automatically or manually
- 🔄 Back up your vault to a Git repository (GitHub, GitLab, etc.)
- 📜 Track the history of every note you've written
- ↩️ Revert changes to any previous version
- 🌐 Sync your vault across multiple devices
- 🔐 Keep your vault secured with automatic backups

## Why Use the Git Plugin?

**Git Plugin turns Obsidian into a professional knowledge management system:**

- **Never lose work** — Every note is backed up, every change is tracked
- **Time travel** — See how your notes evolved, revert to old versions
- **Multi-device sync** — Keep your vault updated across laptop, tablet, phone
- **Automatic backups** — Set it and forget it; backups happen in the background
- **Collaborate safely** — Version control makes working with others worry-free
- **No terminal needed** — Everything works inside Obsidian with buttons and commands

## How It Works

1. **Set up Git** — Initialize your vault as a Git repository (or clone an existing one)
2. **Configure the plugin** — Add your Git server details (GitHub, GitLab, etc.)
3. **Enable auto-backup** — Set it to commit every 10 minutes, hourly, or daily
4. **Work normally** — Your changes are tracked automatically
5. **Sync anytime** — Pull the latest changes from other devices or push your work

## Typical Workflow

```
📝 You edit your notes normally in Obsidian
    ↓

    ↓

    ↓
💻 Other devices pull the latest changes
    ↓
✅ Your vault stays in sync everywhere
```

## Example: Setting Up Automatic Backup

**Enable auto-backup with these steps:**
1. Open Plugin Settings → Obsidian Git
2. Toggle "Automatically commit and push" ON
3. Set interval: `10 minutes` (or your preference)
4. Add a commit message template: `"Auto-save: <%= date %>"`
5. Done! Your vault now backs up automatically

**Result:** Every 10 minutes, your changes are:
- ✅ Committed with a timestamp
- ✅ Pushed to your remote repository
- ✅ Safely backed up in the cloud
